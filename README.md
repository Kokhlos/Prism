# 🦋 Prism

Позволяет быстро обновить текст на скриншотах подменяя ключи в таблице.

## Как пользоваться

**1. Загрузи скриншоты**
Имя каждого файла должно начинаться с числа — это ключ для привязки к строке в таблице переводов. Например: `1_banner.png`, `2_home.png`.

**2. Вставь переводы из Google Sheets**
Открой таблицу → `Ctrl+A` → `Ctrl+C` → вставь в поле и нажми «Применить». Первая строка — названия языков, первая колонка — числовые ключи.

**3. Настрой шрифт**
Загрузи `.ttf`/`.otf` файл и настрой размер, цвет, жирность, обводку и тень. Сохрани настройки как пресет — он запомнит всё включая позиции текста.

**4. Выбери позицию текста**
Нажми на точку в сетке для каждого формата экрана. Каждый формат хранит свою позицию независимо.

**5. Выбери языки и нажми «Сгенерировать»**
Результаты скачиваются одним ZIP-архивом, файлы разложены по папкам языков.

## Пресеты

Пресет сохраняет шрифт + все параметры стиля + позиции текста для всех форматов.

- **Сохранить** — хранится в браузере
- **⬇ Экспорт** — скачать как `.json` файл
- **⬆ Импорт** — загрузить `.json` на другой машине

---

## Разместить онлайн

**Netlify** — самый быстрый способ: перетащи `index.html` на [netlify.com/drop](https://netlify.com/drop).

**GitHub Pages** — создай репозиторий, загрузи файл как `index.html`, включи Pages в Settings.

---

## Технически

Всё работает в браузере — никакие файлы никуда не отправляются. Единственный внешний запрос — опциональная загрузка Google Sheets по URL.

Зависимости: JSZip (архивы) + Google Fonts (интерфейс).
